# personal

DAO слой для IndexedDB на базе Dexie с поддержкой миграций, CRUD-операций, поиска и экспортов/импортов JSON/ZIP.

## Возможности
- Типы данных: `Channel`, `Entry`, `Block`, `Metric`, `Asset` и вспомогательные модели.
- Миграции схемы IndexedDB (3 версии) с индексами для поиска по каналам и датам.
- CRUD-операции и выборки по диапазону дат или каналу.
- Экспорт/импорт снимков в формате JSON или ZIP (`snapshot.json`) с кодированием вложений в base64/Blob.

## Быстрый старт
1. Добавьте зависимости `dexie` и `jszip` (указаны в `package.json`).
2. Импортируйте DAO:

```ts
import { journalDao } from './dist';
```

3. Используйте методы `journalDao` для работы с таблицами или `exportToJson`/`exportToZip` для бэкапов.
